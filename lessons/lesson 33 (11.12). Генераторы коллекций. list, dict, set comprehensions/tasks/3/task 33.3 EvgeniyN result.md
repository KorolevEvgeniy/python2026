### 1. Формулировка задания
Требуется построить словарь `hub_load` с помощью dict comprehension, где:
- Ключ — название хаба (`hub["name"]`).
- Значение — коэффициент загрузки: **сумма зрителей всех потоков** / `capacity`, округлённый до 1 знака.
- В словарь включаются только хабы с ≥2 потоками.
- Исходный список `hubs` не должен изменяться.

**Пример входных данных** (из условия):
```python
hubs = [
    {"name": "DevHub", "capacity": 200, "streams": [{"title": "Python", "viewers": 120}, {"title": "Go", "viewers": 80}]},
    # ... другие хабы
]
```
**Ожидаемый вывод** (из условия):
```python
{'DevHub': 1.0, 'AsyncHub': 1.7}
```

### 4. Ошибки и способы исправления
**Блокирующая ошибка (функциональность):**
- **Неверный расчёт коэффициента.** Вместо `сумма_зрителей / capacity` используется `(сумма_зрителей / кол-во_потоков) / capacity`, что приводит к значениям 0.5 и 0.6 вместо 1.0 и 1.7.  
  **Исправление:**  
  Заменить вычисление на:
  ```python
  sum(stream["viewers"] for stream in hub["streams"]) / hub["capacity"]
  ```

**Минорная ошибка (стиль):**
- Неоптимальное форматирование выражения (разрыв строки в середине генератора).  
  **Исправление:**  
  Упростить код:
  ```python
  result = {
      hub["name"]: round(
          sum(stream["viewers"] for stream in hub["streams"]) / hub["capacity"], 
          1
      )
      for hub in hubs
      if len(hub.get("streams", [])) >= 2
  }
  ```

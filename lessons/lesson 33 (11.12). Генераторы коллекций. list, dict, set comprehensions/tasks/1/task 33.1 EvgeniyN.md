# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» — разобраться, как превращать повторяющиеся циклы в короткие выражения без потери смысла.

## Условие задачи
**Сложность**: <span style="color: #c62828">● сложный</span>. примерное время: 15–20 минут

У тебя есть словарь `stations`, где ключ — название станции метро, а значение — список из трёх чисел: средняя утренняя, дневная и вечерняя загрузка. Сформируй новый список строк через comprehension. Каждая строка содержит название станции и первое время суток, когда загрузка превысила 70%. Если ни один показатель не выше 70, станцию пропусти. Формат строки: `"Тверская → день"`.

## Пример входных данных
```
stations = {
    "Тверская": [65, 82, 77],
    "Савёловская": [55, 60, 58],
    "Бауманская": [71, 69, 88]
}
labels = ["утро", "день", "вечер"]
```

## Пример выходных данных
```
['Тверская → день', 'Бауманская → утро']
```

## Критерии успешного выполнения
- Используется одно выражение list comprehension с проходом по парам станция-значения.
- Для определения первого превышения берётся индекс в изначальном списке `labels`.
- Станции без показателей > 70 не попадают в результат.
- Формат записи станции и времени точный: `Название → период`.

## Подсказки
- Внутри comprehension можно распаковывать пары через `for name, loads in stations.items()`.
- Чтобы найти первый индекс, используй дополнительный генератор внутри `next()`.

## Частые ошибки и ограничения
- Нельзя добавлять вложенный цикл `for` в теле без контроля; важно сохранить читаемость.
- Не используйте `append`. результат должен создаваться comprehension-ом.
- Не меняй исходный словарь и список `labels`.

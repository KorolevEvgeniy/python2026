Задание 2. Основы Python. Переменные и типы данных

### Описание
Напиши программу, которая просит пользователя ввести название курса, продолжительность в часах и стоимость одного часа. Приведи продолжительность к int, стоимость к float и вычисли общую стоимость курса. Покажи каждое введенное значение и общий расчет с указанием типов.

### Пример входных данных
```
Python Base
32
599.5
```

### Пример выходных данных
```
Курс: Python Base (str)
Часы: 32 (int)
Стоимость часа: 599.5 (float)
Полная стоимость: 19184.0 руб. (float)
```

### Критерии успешного выполнения
- Все значения читаются через input() по порядку: название, часы, стоимость часа.
- Часы преобразуются к int, стоимость к float.
- Итоговая стоимость вычисляется как произведение часов и стоимости.
- В выводе отражены исходные значения и законченная строка с результатом.
- Программа завершается без ошибок при корректном вводе.

### Подсказки
- Проверяй, что преобразование выполняется сразу после чтения значения.
- Используй метод format() или f-строки, чтобы собрать итоговый текст.
- Для типов применяй type(...)__name__.

### Частые ошибки и ограничения
- Не путай порядок чтения чисел. иначе преобразование завершится ошибкой.
- Не добавляй сторонние функции округления. оставь стандартное поведение float.
- Не сохраняй результат в виде строки до окончания вычислений.

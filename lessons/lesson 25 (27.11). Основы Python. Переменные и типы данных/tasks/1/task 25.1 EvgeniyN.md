Задание 1. Основы Python. Переменные и типы данных

### Описание
Сделай программу, которая читает из input() имя, возраст и уровень владения Python. Оставь имя строкой, возраст приведи к int, уровень приведи к float. Выведи каждое значение и его тип отдельной строкой.

### Пример входных данных
```
Evgeniy
17
3.5
```

### Пример выходных данных
```
Имя: Evgeniy (str)
Возраст: 17 (int)
Уровень: 3.5 (float)
```

### Критерии успешного выполнения
- Используются три вызова input() подряд.
- Возраст преобразуется к int без ошибок.
- Уровень преобразуется к float.
- Все строки вывода содержат подписи и типы значений.
- Программа завершается без исключений при корректном вводе.

### Подсказки
- Сохраняй результат type(...)__name__ в переменной для повторного использования.
- Применяй f-строки, чтобы объединять подписи и значения.
- Проверь порядок преобразований, чтобы не вызвать ValueError.

### Частые ошибки и ограничения
- Не пытайся складывать строку и число без приведения типов.
- Не меняй порядок ввода данных.
- Не добавляй дополнительные сообщения, кроме требуемых строк.

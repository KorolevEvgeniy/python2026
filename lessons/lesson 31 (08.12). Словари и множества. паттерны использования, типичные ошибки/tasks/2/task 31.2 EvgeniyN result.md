#### 1. Формулировка задания
**Цель:** Накопить количество поездок по станциям в словаре `rides` из списка `logs`, безопасно удалить станции из списка `finished` (без ошибок при отсутствии ключа) и вывести обновлённый словарь.  
**Ключевые требования:**
- Использовать `get`/`setdefault` для накопления значений.
- Удалять записи через `pop(key, запасное_значение)`.
- Вывести словарь после удаления.
- Обрабатывать случаи, когда ключи из `finished` отсутствуют в словаре.

**Пример входных данных:**
```python
logs = [('park', 12), ('river', 7), ('park', 5), ('mall', 3)]
finished = ['mall']
```

#### 6. Вариант полного решения
```python
logs = [('park', 12), ('river', 7), ('park', 5), ('mall', 3)]
finished = ['mall']
rides = {}

# Накопление поездок
for name, count in logs:
    rides[name] = rides.get(name, 0) + count

# Безопасное удаление станций
for finish in finished:
    rides.pop(finish, None)  # Запасное значение не обязательно выводить

# Итоговый вывод
print(rides)
```

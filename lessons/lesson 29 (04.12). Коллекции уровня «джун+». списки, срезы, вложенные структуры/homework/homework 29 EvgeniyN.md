### Блок 1: Работа со списками

Ты готовишься к походу в горы. Создай список обязательного снаряжения, добавь к нему дополнительные предметы из списка `дополнительное_снаряжение`, выведи общее количество предметов, покажи первый и последний предмет, а затем выведи нумерованный список всего снаряжения.

Пример входных данных:
```python
base_gear = ["палатка", "спальник", "коврик"]
additional_gear = ["фонарик", "аптечка", "карта"]
```

### Блок 2: Срезы

У тебя есть список высот точек маршрута за день (в метрах). 
1. Выведи первые три высоты.
2. Выведи последние три высоты.
3. Выведи каждую вторую высоту, начиная с первой.
4. Выведи список без первой и последней высоты.

Пример входных данных:
```python
heights = [1200, 1350, 1400, 1100, 1250, 1300, 1450, 1500, 1350]
```

### Блок 3: Мутация списков

У тебя есть список дел по подготовке к походу. 
1. Добавь в конец списка дела: "проверить запас воды" и "упаковать еду".
2. Вставь в середину списка (после второго элемента) дело "зарядить гаджеты".
3. Удали из списка дело "купить билеты" (если оно есть).
4. Перемести последнее дело в начало списка.

Пример входных данных:
```python
tasks = ["купить билеты", "собрать рюкзак", "запланировать маршрут"]
```

### Блок 4: Вложенные структуры

У тебя есть список участников похода, каждый участник представлен словарем с полями: `имя`, `роль`, `вес_рюкзака` (в кг). 
1. Распечатай информацию в формате: `Имя: роль (вес_рюкзака кг)`.
2. Сформируй список имен участников с весом рюкзака больше 15 кг.
3. Создай новый список только с гидами (роль "гид").
4. Подсчитай средний вес рюкзака.

Пример входных данных:
```python
participants = [

    {"имя": "Мария", "роль": "участник", "вес_рюкзака": 12},
    {"имя": "Иван", "роль": "гид", "вес_рюкзака": 18},
    {"имя": "Елена", "роль": "участник", "вес_рюкзака": 10}
]
```

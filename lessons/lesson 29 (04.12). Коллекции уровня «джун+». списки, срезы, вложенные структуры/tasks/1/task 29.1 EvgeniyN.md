# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. В этом блоке важно освоить создание списков, обращение по индексу и перебор, чтобы уверенно работать с упорядоченными данными.

## Условие задачи
Тебе нужно подготовить минимальный набор кистей и красок для выездного пленэра. Создай список стартовых предметов, дополни его экстренными запасами, выведи количество вещей, покажи элементы под номером 2 и последний. Затем напечатай список с порядковыми номерами, чтобы не забыть ничего дома.

## Пример входных данных
```python
base_set = ["кисть круглая", "кисть плоская", "гуашь"]
extras = ["акварель", "палитра"]
```

## Пример выходных данных
```
Всего позиций: 5
Второй предмет: кисть плоская
Последний предмет: палитра
1. кисть круглая
2. кисть плоская
3. гуашь
4. акварель
5. палитра
```

## Критерии успешного выполнения
- Список создается из готовых данных и расширяется новым списком
- Используются индексы для доступа к элементам без циклов
- Количество предметов выводится функцией `len`
- Итоговый список выводится с нумерацией через цикл

## Подсказки
- Добавь дополнительные предметы через `extend`, чтобы не перебирать их вручную
- Используй положительный индекс для второго элемента, отрицательный для последнего
- Функция `enumerate` упростит одновременный вывод номера и названия

## Частые ошибки и ограничения
- Не перезаписывай исходный список строкой, работай только со списковыми методами
- Следи, чтобы список не оставался пустым перед обращением к индексу 1
- Не добавляй новые элементы через конкатенацию строк, это не список
- Проверяй, что цикл печатает актуальное содержимое после всех изменений

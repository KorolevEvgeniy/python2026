# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Нужно научиться уверенно доставать разные части списка, использовать отрицательные индексы и шаг, чтобы не писать дополнительные циклы.

## Условие задачи
Ты анализируешь список продаж кофе по дням. Требуется:
1. Получить первые пять значений для отчета руководителю.
2. Собрать последний рабочий блок недели с помощью отрицательных индексов.
3. Выбрать каждый третий элемент, чтобы отследить тренд.
4. Создать копию списка без первого и последнего элементов.
Выведи все результаты с понятными подписями.

## Пример входных данных
```python
sales = [23, 21, 25, 30, 27, 29, 31, 28, 26, 33]
```

## Пример выходных данных
```
Начало недели: [23, 21, 25, 30, 27]
Хвост недели: [29, 31, 28, 26, 33]
Каждый третий: [25, 27, 31, 33]
Без крайних: [21, 25, 30, 27, 29, 31, 28, 26]
```

## Критерии успешного выполнения
- Все выборки сделаны срезами, без функций и циклов
- Использованы как положительные, так и отрицательные границы
- Копия списка создаётся через `[:]`
- Каждое требование сопровождается отдельным выводом

## Подсказки
- Чтобы получить последние n значений, оставь пустым старт и укажи отрицательный `stop`
- Параметр `step` можно использовать для пропуска элементов
- Не забывай, что `stop` не включается в срез
- Срез `list[1:-1]` удобно удаляет крайние элементы

## Частые ошибки и ограничения
- Не меняй исходный список при формировании отчёта
- Следи, чтобы шаг не был равен нулю
- Не перепутай местами `start` и `stop`
- Проверяй протяженность списка перед выборкой, чтобы не получить пустой срез

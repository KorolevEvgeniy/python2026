# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Тренируем чтение списка словарей и безопасное извлечение значений по нескольким уровням.

## Условие задачи
Ты модерируешь онлайн курс и ведешь список групповых встреч:
1. Создай список словарей с ключами `week`, `meeting`, `topic`, `students` (список имен).
2. Выведи каждую встречу строкой `Неделя X. Тема — Y. Участники: ...`.
3. Собери отдельный список всех уникальных студентов через проход по вложенным спискам.
4. Построй словарь вида `topic -> количество участников`, используя длину вложенного списка `students`.
5. Отфильтруй встречи, в которых участвует студент с именем из переменной `focus_student`.

## Пример входных данных
```python
sessions = [
    {"week": 1, "meeting": "Zoom", "topic": "синхронизация", "students": ["Аня", "Кирилл"]},
    {"week": 2, "meeting": "Discord", "topic": "проект", "students": ["Кирилл", "Марк", "Лиза"]}
]
focus_student = "Кирилл"
```

## Пример выходных данных
```
Неделя 1. Тема. синхронизация. Участники. Аня, Кирилл
Неделя 2. Тема. проект. Участники. Кирилл, Марк, Лиза
Все студенты. {'Аня', 'Кирилл', 'Лиза', 'Марк'}
Размер групп по темам. {'синхронизация': 2, 'проект': 3}
Сессии с Кирилл. [{'week': 1, 'meeting': 'Zoom', 'topic': 'синхронизация', 'students': ['Аня', 'Кирилл']}, {'week': 2, 'meeting': 'Discord', 'topic': 'проект', 'students': ['Кирилл', 'Марк', 'Лиза']}]
```

## Критерии успешного выполнения
- В расписании используются списки словарей и вложенные списки студентов
- Вывод по неделям формируется циклом без ручного обращения к индексам
- Словарь `topic -> количество` собирается программно
- Фильтрация по студенту не изменяет исходные данные

## Подсказки
- Для копирования вложенного списка используйте `students.copy()` при необходимости
- Уникальные имена удобно хранить в `set`
- Чтобы проверить участие студента применяй оператор `in` к подсписку

## Частые ошибки и ограничения
- Не модифицируй список студентов во время перебора
- Проверяй наличие всех ключей в словаре перед обращением
- Не хардкоди количество встреч. цикл сам обходится
- Не превращай `set` обратно в список без причины

# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Учимся уверенно добавлять, вставлять и удалять элементы, а также менять их порядок.

## Условие задачи
Ты собираешь набор кистей для серии миниатюр:
1. Создай список основных кистей.
2. Добавь в конец экспериментальные инструменты с помощью `extend`.
3. Вставь кисть для деталей в позицию №2.
4. Удали из списка кисть, которая испортилась, по значению.
5. Перемести последнюю кисть в начало, используя `pop` + `insert`.
6. Выведи итоговый порядок, чтобы свериться перед поездкой.

## Пример входных данных
```python
brushes = ["круглая №1", "плоская №6", "веерная"]
extras = ["синтетика тонкая", "щетина широкая"]
```

## Пример выходных данных
```
['щетина широкая', 'круглая №1', 'детальная №0', 'плоская №6', 'веерная', 'синтетика тонкая']
```

## Критерии успешного выполнения
- Использованы минимум три разных метода мутации (`append/extend`, `insert`, `remove`, `pop`)
- Перемещения выполняются без ручного обмена переменных
- Итоговый список отражает все изменения
- Нет дублирования логики через создание новых списков

## Подсказки
- После `extend` список сразу содержит новые элементы
- Для вставки на вторую позицию используй индекс `1`
- Перед `remove` убедись, что такая кисть точно есть
- Перемещение последнего элемента: `item = brushes.pop()` → `brushes.insert(0, item)`

## Частые ошибки и ограничения
- Не вызывай `remove` без проверки наличия элемента
- Не модифицируй список внутри цикла перебора
- Следи, чтобы индекс вставки не выходил за пределы
- Не забывай распечатать итоговую версию списка

### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает привести список предметов к аккуратному виду.

Требования:
- прочитать строку с названиями предметов, разделенными запятой
- удалить пробелы вокруг каждого названия
- привести все названия к нижнему регистру
- исключить повторяющиеся названия
- отсортировать итоговый список по алфавиту и вывести его на экран

### Пример входных данных
```
  Математика,   Информатика, физика, информатика,  математика 
```

### Пример выходных данных
```
['информатика', 'математика', 'физика']
```

### Критерии успешного выполнения
- строка корректно разбивается по запятой
- пробелы вокруг названий удалены
- дубликаты предметов убраны
- итоговый список отсортирован по алфавиту в нижнем регистре

### Подсказки
- используй `split(',')` и `strip()`
- для удаления дубликатов можно использовать `set`, но помни, что он не сохраняет порядок
- сортировку делай с помощью `sorted` или метода `sort`

### Частые ошибки и ограничения
- не опирайся на исходный порядок после приведения к множеству
- учитывай, что список может быть пустым
- не используй внешние библиотеки . стандартных средств достаточно

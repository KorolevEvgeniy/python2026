### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая получает на вход строку с именем и отчеством студента и нормализует ее.

Требования:
- убрать лишние пробелы по краям
- разделить строку на два слова
- привести каждое слово к виду `ПерваяБукваЗаглавная` остальные строчные
- вывести общую длину строки без пробелов и отдельно длину каждого слова

### Пример входных данных
```
  евгений   николаевич  
```

### Пример выходных данных
```
Имя: Евгений
Отчество: Николаевич
Длина имени: 7
Длина отчества: 11
Общая длина без пробелов: 18
```

### Критерии успешного выполнения
- строка очищается от лишних пробелов по краям
- корректно выделяются два слова
- регистр букв приводится к требуемому виду
- длины считаются без лишних пробелов
- программа устойчива к разному количеству пробелов между словами

### Подсказки
- используй `strip` и `split`
- контролируй количество полученных элементов после `split`
- для длины применяй `len`

### Частые ошибки и ограничения
- не делай жесткой привязки к одному пробелу между словами
- обрабатывай ситуацию, когда введено меньше или больше двух слов
- не используй внешние библиотеки

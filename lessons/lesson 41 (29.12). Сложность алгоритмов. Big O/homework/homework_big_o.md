A: O(n) - В цикле выполняются только операции сравнения и присваивания с постоянным временем O(1). Общее количество итераций равно n, где n — количество элементов.

B: O(n^2) - Алгоритм использует два вложенных цикла. Внешний цикл проходит по n элементам. Для каждого i внутренний цикл проходит от i+1 до n, что в среднем дает около n/2 итераций. Общее количество операций пропорционально n*(n-1)/2, что упрощается до O(n²).
 
C: O(n log n) - Сначала выполняется сортировка списка с помощью sorted(), которая имеет сложность O(n log n). Затем выполняется один проход по отсортированному списку с сложностью O(n). Поскольку O(n log n) доминирует над O(n), общая сложность составляет O(n log n).
